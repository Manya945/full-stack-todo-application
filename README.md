# full-stack-todo-application
1) objective
   The objective of this project is to develop a full-stack web application that allows users to manage their daily tasks efficiently.
2) project description
   This Todo Application allows users to add, update, delete, and mark tasks as completed using a MERN stack architecture.
3) features
 1. Add tasks
2. Update tasks
3. Delete tasks
4. Mark tasks as completed
5. Real-time updates

4) technologies used
   1. Frontend: React.js
2. Backend: Node.js, Express.js
3. Database: MongoDB

 5) API endpoints
    1. GET /tasks - Fetch all tasks
2. POST /tasks - Add task
3. PUT/tasks/:id - Update task 
4. DELETE /tasks/:id - Delete task
